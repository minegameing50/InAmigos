# InAmigos Foundation - Business Website

A professional, responsive business website for InAmigos Foundation built with HTML, CSS, and JavaScript.

## 📋 About

InAmigos Foundation is an NGO dedicated to social impact and community engagement. This website serves as the digital presence to promote awareness about our mission, showcase our projects, demonstrate our social impact, and invite volunteers to join our cause.

## 🌐 Website Features

- **Home Page** - Professional introduction to InAmigos Foundation
- **Mission & Vision** - Clear communication of organizational goals and values
- **Projects** - Showcase of ongoing and completed initiatives
- **Social Impact** - Measurable results and community achievements
- **Get Involved** - Volunteer opportunities and engagement pathways
- **Contact** - Easy ways for stakeholders to reach out
- **Responsive Design** - Seamless experience across all devices and screen sizes

## 🛠️ Technology Stack

- **HTML** (46.8%) - Semantic website structure and content
- **CSS** (38.8%) - Professional styling and responsive layout
- **JavaScript** (14.4%) - Interactive features and user engagement

## 🚀 Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/minegameing50/InAmigos.git
   ```

2. Navigate to the project:
   ```bash
   cd InAmigos
   ```

3. Open `index.html` in your web browser
